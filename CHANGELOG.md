# Changelog for peridio 0.3

Peridio CLI 0.3 introduces new commands to support managing CA certificates and their JITP configurations.

## 0.3.0

### Enhancements

- Add `peridio ca-certificates`.
  - Add `create`.
  - Add `delete`.
  - Add `get`.
  - Add `list`.
  - Add `update`.
  - Add `create-verification-code`.
